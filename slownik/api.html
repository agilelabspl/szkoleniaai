API (Application Programming Interface): co to jest i po co AI potrzebuje API?

API to sposób, w jaki programy rozmawiają ze sobą. Dzięki API możesz podłączyć ChatGPT do swojej aplikacji, Excela, strony czy systemu.

Wyobraź sobie...

Restauracja. Ty (aplikacja) nie wchodzisz do kuchni (serwer AI). Mówisz kelnerowi (API) co chcesz, on idzie do kuchni, przynosi zamówienie.

Nie musisz wiedzieć jak działa kuchnia, znać przepisów ani mieć kuchennego sprzętu.

API robi to samo. Twoja aplikacja wysyła pytanie przez API do AI, AI odpowiada, aplikacja pokazuje Ci wynik. Cała magia dzieje się „w kuchni", Ty widzisz tylko gotowe danie.

Gdzie to spotkasz?

  • Chatboty na stronach firm – łączą się z AI przez API, żeby odpowiadać klientom w czasie rzeczywistym
  • Automatyzacje (Make, Zapier) – łączą różne narzędzia przez API: mail + AI + arkusz kalkulacyjny
  • Aplikacje mobilne z AI – każda appka z funkcją „zapytaj AI" korzysta pod spodem z API
  • GitHub Copilot – podpowiadanie kodu w edytorze, które łączy się z GPT przez API

Najczęstsze pytania o API

Czy muszę umieć programować żeby używać API?

Bezpośrednio z API, tak, potrzeba podstaw. Ale narzędzia no-code (Make, Zapier, n8n) pozwalają korzystać z API bez programowania. Łączysz bloczki jak LEGO, „weź maila, wyślij do ChatGPT, zapisz odpowiedź w Google Sheet".

Ile kosztuje API do AI?

Zależy od modelu i ilości użycia. Przykład: GPT-4o od OpenAI kosztuje ok. 10 zł za milion tokenów wejściowych (to ok. 750 000 słów). Dla małej firmy, zwykle 20-100 zł/miesiąc. Dużo taniej niż miesięczna subskrypcja ChatGPT Plus.

Do czego mogę podłączyć API AI?

Do prawie wszystkiego: strony internetowej, Excela/Google Sheets, systemu CRM, maila, Slacka, aplikacji mobilnej, systemu fakturowania. Jeśli coś ma możliwość łączenia się z internetem, prawdopodobnie możesz podłączyć do niego AI.
