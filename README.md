# AI Szkolenia Hub

Katalog szkoleń AI w Polsce.

## Struktura projektu

```
ai-szkolenia-hub/
├── index.html      # Główna strona (białe tło, SEO zoptymalizowana)
├── robots.txt      # Instrukcje dla crawlerów
├── sitemap.xml     # Mapa strony dla wyszukiwarek
└── README.md       # Ten plik
```
